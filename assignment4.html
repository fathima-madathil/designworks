PROGRAMMING LANGUAGES

A programming language is a formal language designed to communicate instructions to a machine,particularly a computer.Programming languages can be used to create programs that control the behaviour of a machine and/or to express algorithms precisely.

c++

c++(prounounced c plus plus)is a programming language that is general purpose,statically typed free form,multi paradigm and compiled.it is considered as an intermediate language as it comprises both high level and low level language features

c

c is one of the most widely used programming languages of all time,and c compilers are available for the majority of computer architectures and operating systems.

java

java is a computer programming language that is concurrent,class based,object oriented and specifically designed to have a few implementaion dependencies as possible.it is intented to let application developers "write once,run anywhere (WORA),meaning that code that runs on one platform does not need to be recompiled to run on another
